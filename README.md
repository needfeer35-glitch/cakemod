# 🎂 CakeAuto — Fabric Mod для Minecraft 1.21.5

Мод для автоматического создания тортов!

## Возможности
- Команда `/tort` — открывает меню настроек
- Галочка "Авто-торт" — автоматически заполняет верстак ингредиентами после каждого созданного торта
- Кнопка открытия верстака прямо из меню

## Рецепт торта
```
[Молоко] [Молоко] [Молоко]
[Сахар]  [Яйцо]  [Сахар]
[Пшеница][Пшеница][Пшеница]
```

## Установка
1. Скачай файл `cakeauto-1.0.0.jar` из вкладки **Actions → последний билд → Artifacts**
2. Положи в папку `.minecraft/mods/`
3. Нужен [Fabric Loader](https://fabricmc.net/use/installer/) и [Fabric API](https://modrinth.com/mod/fabric-api)

## Сборка
```bash
./gradlew build
```
Готовый JAR будет в `build/libs/`
